typescript-1基本类型

Created2023-06-08|Updated2023-06-08|前端
|Post View:

📒 原始数据类型 都要采用小写的类型， 大写类型用来描述实例（包装类）
string number boolean null undefined 枚举 元组 数组 never void object symbol bigint

any 不进行类型检测 如果变量声明时没有赋值默认也是 any 能不写就不写 写多了就变成了 anyscript
